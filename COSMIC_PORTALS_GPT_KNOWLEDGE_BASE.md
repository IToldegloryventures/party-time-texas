## üìã **BUSINESS OVERVIEW**

### **What is Cosmic Portals?**

Cosmic Portals is a multi-tenant SaaS platform that enables businesses and organizations to create NFC-powered engagement experiences. It combines NFC device management, custom landing pages, event management, and analytics into one integrated solution.

### **Core Value Proposition**

- **For Businesses**: Create interactive NFC experiences for customers without technical expertise
- **For Event Organizers**: Manage events with NFC check-ins, custom landing pages, and attendee tracking
- **For Organizations**: Build branded digital experiences that drive engagement and collect data

### **Target Markets**

1. **B2B Business Clients**: Companies wanting NFC engagement for marketing/events
2. **B2C Event Clients**: Event planners, venues, party organizers
3. **Educational Institutions**: Schools and universities for campus engagement
4. **Enterprise**: Large organizations needing custom branded solutions

## üèóÔ∏è **TECHNICAL ARCHITECTURE**

### **Frontend Stack**

- **Framework**: Next.js 15 with React 19
- **Styling**: Tailwind CSS with custom component library
- **State Management**: React hooks and context
- **Authentication**: Clerk (handles sign-in, sign-up, user management)
- **Deployment**: Vercel (automatic deployments from GitHub)

### **Backend Stack**

- **API**: Next.js API routes (serverless functions)
- **Database**: Supabase (PostgreSQL with real-time features)
- **Security**: Row Level Security (RLS) for multi-tenant data isolation
- **Payments**: Stripe (subscription management, billing)
- **AI Integration**: Google Gemini API (content generation, insights)

### **Database Schema**

**Core Tables**:

- `users` - User accounts with roles and permissions
- `organizations` - Multi-tenant organization data
- `nfc_devices` - NFC device registration and management
- `events` - Event creation and management
- `attendees` - Event attendance tracking
- `landing_pages` - Custom landing page content
- `nfc_scans` - Scan tracking and analytics
- `tasks` - Task management system

**Security**: Row Level Security policies ensure data isolation between organizations

### **Key Technical Features**

- **Multi-tenant Architecture**: Complete data isolation between organizations
- **Role-based Access Control**: Granular permissions system
- **Real-time Updates**: Live data synchronization
- **API-first Design**: RESTful APIs for all functionality
- **Responsive Design**: Mobile-first UI/UX
